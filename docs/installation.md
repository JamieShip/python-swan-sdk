## Table Of Contents

## Create Virtual Environment

MacOS and Linux
```bash
python3 -m venv .venv
source .venv/bin/activate
```

## Install via PyPI
```bash
pip install swan-sdk
```

## Clone from Github
```bash
git clone https://github.com/swanchain/orchestrator-sdk.git
git checkout dev
```
